Paso 1: Modelo de regresi√≥n lineal

Donde necesitamos encontrar los coeficientes a (intercepto) y b (pendiente).

Paso 2: F√≥rmulas de m√≠nimos cuadrados
b = (n‚àëxy - ‚àëx‚àëy) / (n‚àëx¬≤ - (‚àëx)¬≤)

Paso 5: Coeficiente de determinaci√≥n R¬≤

Calculamos R¬≤ para evaluar la bondad del ajuste:

R¬≤ = 1 - (SSres / SStot)

Donde SSres es la suma de cuadrados residuales y SStot es la suma total de cuadrados.

Paso 6: Estimaci√≥n a los 25 minutos

Paso 7: Comentarios sobre el modelo

El modelo lineal es muy adecuado para estos datos porque:

  • Los puntos siguen una tendencia claramente lineal
  • R¬≤ es muy cercano a 1, indicando un excelente ajuste
  • La relaci√≥n temperatura-tiempo es f√≠sicamente coherente

import numpy as np import matplotlib.pyplot as plt from sklearn.linear_model import LinearRegression from sklearn.metrics import r2_score from scipy.optimize import curve_fit # Datos del ejercicio 3 tiempo = np.array([0, 2, 4, 6, 8]) temperatura = np.array([100, 81.9, 67.0, 54.8, 44.8]) # M√©todo 1: Regresi√≥n exponencial mediante linearizaci√≥n def regresion_exponencial_manual(x, y): # Transformaci√≥n logar√≠tmica ln_y = np.log(y) n = len(x) sum_x = np.sum(x) sum_ln_y = np.sum(ln_y) sum_x_ln_y = np.sum(x * ln_y) sum_x2 = np.sum(x**2) # Calcular coeficientes de la forma ln(y) = A + b*x b = (n * sum_x_ln_y - sum_x * sum_ln_y) / (n * sum_x2 - sum_x**2) A = (sum_ln_y - b * sum_x) / n a = np.exp(A) return a, b # Calcular regresi√≥n manual a_manual, b_manual = regresion_exponencial_manual(tiempo, temperatura) # M√©todo 2: Usar sklearn con datos linearizados ln_temperatura = np.log(temperatura) X = tiempo.reshape(-1, 1) modelo_exp = LinearRegression() modelo_exp.fit(X, ln_temperatura) a_sklearn = np.exp(modelo_exp.intercept_) b_sklearn = modelo_exp.coef_[0] # M√©todo 3: Usar scipy.optimize.curve_fit (no lineal) def modelo_exponencial(x, a, b): return a * np.exp(b * x) popt, pcov = curve_fit(modelo_exponencial, tiempo, temperatura) a_scipy, b_scipy = popt # Predicciones y_pred_manual = a_manual * np.exp(b_manual * tiempo) y_pred_sklearn = a_sklearn * np.exp(b_sklearn * tiempo) y_pred_scipy = modelo_exponencial(tiempo, a_scipy, b_scipy) # Calcular R¬≤ para cada m√©todo r2_manual = r2_score(temperatura, y_pred_manual) r2_sklearn = r2_score(temperatura, y_pred_sklearn) r2_scipy = r2_score(temperatura, y_pred_scipy) # Estimaci√≥n a los 10 minutos tiempo_10 = 10 temp_10_manual = a_manual * np.exp(b_manual * tiempo_10) temp_10_sklearn = a_sklearn * np.exp(b_sklearn * tiempo_10) temp_10_scipy = modelo_exponencial(tiempo_10, a_scipy, b_scipy) # Resultados print("=== EJERCICIO 3: REGRESI√ìN EXPONENCIAL ===") print("\nM√©todo 1 - Sklearn (linearizaci√≥n):") print(f"Ecuaci√≥n: y = {a_sklearn:.3f} * exp({b_sklearn:.4f} * x)") print(f"R¬≤ = {r2_sklearn:.6f}") print(f"Temperatura a los 10 min: {temp_10_sklearn:.2f}¬∞C") print("\nM√©todo 3 - Scipy curve_fit (no lineal):") print(f"Ecuaci√≥n: y = {a_scipy:.3f} * exp({b_scipy:.4f} * x)") print(f"R¬≤ = {r2_scipy:.6f}") print(f"Temperatura a los 10 min: {temp_10_scipy:.2f}¬∞C") # Crear gr√°fico comparativo plt.figure(figsize=(15, 5)) # Gr√°fico 1: Datos originales con ajuste exponencial plt.subplot(1, 3, 1) plt.scatter(tiempo, temperatura, color='red', s=100, label='Datos observados', zorder=5) # Curva suave para la regresi√≥n exponencial tiempo_smooth = np.linspace(0, 12, 100) temp_smooth_manual = a_manual * np.exp(b_manual * tiempo_smooth) temp_smooth_scipy = modelo_exponencial(tiempo_smooth, a_scipy, b_scipy) plt.plot(tiempo_smooth, temp_smooth_manual, 'b-', linewidth=2, label=f'Manual: y = {a_manual:.1f}e^({b_manual:.3f}x)') plt.plot(tiempo_smooth, temp_smooth_scipy, 'g--', linewidth=2, label=f'Scipy: y = {a_scipy:.1f}e^({b_scipy:.3f}x)') plt.scatter([tiempo_10], [temp_10_manual], color='blue', s=150, label=f'Pred. 10 min: {temp_10_manual:.1f}¬∞C', zorder=5) plt.xlabel('Tiempo (minutos)') plt.ylabel('Temperatura (¬∞C)') plt.title('Regresi√≥n Exponencial - üîç Criterios de Selecci√≥n del Modelo

1. Regresi√≥n Lineal

Cu√°ndo usar: Cuando existe una relaci√≥n proporcional directa entre variables.

Ventajas: Simple, interpretable, c√°lculos directos.

Ejemplo: Calentamiento constante, relaciones f√≠sicas lineales.

2. Regresi√≥n Logar√≠tmica

Cu√°ndo usar: Crecimiento/decaimiento que se desacelera con el tiempo.

Ventajas: Buena para procesos que tienden a una as√≠ntota.

Ejemplo: Concentraciones que disminuyen, aprendizaje, saturaci√≥n.

3. Regresi√≥n Exponencial

Cu√°ndo usar: Crecimiento/decaimiento exponencial.

Ventajas: Ideal para procesos naturales exponenciales.

Ejemplo: Enfriamiento, crecimiento poblacional, decaimiento radioactivo.
